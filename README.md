# Sukoon Veda Ecommerce Website

Premium Sukoon Veda ecommerce storefront built with React and Vite.

## Project Structure

- `index.html` - Vite HTML entry with SEO metadata and JSON-LD.
- `src/main.jsx` - React mount entry.
- `src/App.jsx` - Main ecommerce page composition and cart/checkout state.
- `src/App.css` - Global responsive luxury black-and-gold styling.
- `src/components/` - Reusable storefront sections.
- `src/assets/` - Product and brand image assets.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
